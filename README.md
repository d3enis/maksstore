# MaksStore Web Shop
This web application was my final assignment in high school.
It is meant to be a web shop with features of : 
- Previewing products 
- Adding new products
- Searching through products by name or category
- Cart system (add,remove,checkout,etc.)
- Login system
## Technologies used
- React.Js
- Node.Js
- OracleMySQL 
- Bootstrap 5
- Heroku hosting
